# Tech Stack

## Frontend
- **React 18** with **TypeScript** - UI framework
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **TanStack Query** - Data fetching and caching
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI component library (built on Radix UI)
- **Lucide React** - Icon library

## Backend & Database
- **Supabase** - Backend-as-a-Service
  - **PostgreSQL** - Database with Row Level Security (RLS)
  - **Supabase Auth** - Authentication
  - **Supabase Storage** - File storage for documents
  - **Database Functions** - Server-side logic (PL/pgSQL)

## Development Tools
- **TypeScript** - Type safety
- **ESLint** - Code linting
- **PostCSS** - CSS processing

## AI & Coding Tools
- **Cursor** - AI-powered code editor with Auto agent
- **GitHub Copilot** (via Cursor integration) - AI pair programming

## Key Libraries
- **@supabase/supabase-js** - Supabase client library
- **react-hook-form** + **zod** - Form handling and validation
- **date-fns** - Date manipulation
- **jszip** - File compression for large file uploads
- **mammoth** - Word document processing
- **xlsx** - Excel file processing
